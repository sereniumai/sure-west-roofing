## Design Standards

- Use Tailwind CSS for all styling
- Keep design modern, clean and professional
- This is a roofing company website — prioritise trust, clarity and strong CTAs
- Sure West design system (navy #1B3558, gold #C49A2C, off-white #F7F5F0, Oswald + Inter)

## Git Workflow

- Only work locally unless explicitly asked to push to GitHub
- Do not commit, push, or create branches without being asked
- Wait for confirmation before any Git operation
